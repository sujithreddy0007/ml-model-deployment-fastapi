# 🔮 Machine Learning Model Deployment with FastAPI

This project demonstrates how to train and deploy a Machine Learning model using **FastAPI**, exposing both an API and an HTML form for predictions.

---

## 📌 Features

- Train a classification model using scikit-learn
- Expose REST API with FastAPI
- Render HTML form for input and display prediction
- Swagger UI for API testing
- Deployable on Render or any cloud platform
